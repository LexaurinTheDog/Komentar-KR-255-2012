# Komentář k zákonu č. 255/2012 Sb., o kontrole (kontrolní řád)

Odborný komentář k **kontrolnímu řádu** — obecnému procesnímu předpisu pro kontrolu vykonávanou orgány veřejné moci, na který subsidiárně odkazují kontrolní ustanovení řady zvláštních zákonů. Komentář pokrývá celé normativní jádro (ČÁST PRVNÍ, § 1–30).

- **Rozsah:** ~9 tis. slov, 2 kapitoly + úvod + index.
- **Témata:** zahájení kontroly, práva a povinnosti kontrolujícího a kontrolované osoby, protokol, námitky, ochrana oznamovatele, přestupky, subsidiarita správního řádu.
